# Mini Budget Tracker

A simple, interactive web application built with **HTML, CSS, and ES6 JavaScript** to help users track expenses, savings goals, and visualize their financial progress. Designed as a lightweight portfolio project showcasing modular code, async concepts, and polished UI/UX.

---

## 🚀 Features
- **Add & Manage Expenses**  
  Track spending across categories (e.g., food, transport, entertainment).

- **Savings Goals**  
  Set multiple goals (holiday, car, bond, credit card) and monitor progress.

- **Progress Visualization**  
  Dynamic progress bars and charts to show how close you are to each goal.

- **Responsive Layout**  
  Clean, mobile‑friendly design with CSS enhancements.

- **Persistent Data**  
  Local storage integration ensures your data remains after page reloads.

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** for styling and responsive design  
- **JavaScript (ES6)** for logic, modular functions, and interactivity
